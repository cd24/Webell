Welcome to the Hasklettes example site!

This website will show off some of the rendering powers of hasklettes! It's a simple page, admittedly, play with it!

Custom Elements

You can use custom HTML elements, as long as you provide some helper functions!

divTag [bootstrap_column "md" 12] [ mdl_card "My Card Header" "Some Card Body" [], br, mdl_card "Second Card" "Second Card Body" [("My Action", "http://google.com")] ]

Select

Hasklettes can help you quickly write otherwise unruly select statements. Simply pass it an array of options, and it will handle all the boiler plate for you!

divTag [bootstrap_column "md" 12] [ select [bootstrap_column "md" 5] [ "First Option", "Second Option", "Third Option" ], divTag [bootstrap_column "md" 2] [], select [bootstrap_column "md" 5] [ "Fourth Option", "Fifth Option", "Sixth Option" ] ]

Tables

table ([[1..10], [11..20], [21..30]] :: [[Int]]) [TO "border" "1", bootstrap_column "md" 8]
